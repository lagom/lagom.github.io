Using Lagom with Java

You can use Maven or sbt to build apps you create with the Lagom Java API. If you aren’t using one of these tools already, see these links to help you choose:

  • sbt
  • Maven

See detailed instructions for help setting up the prerequisites and getting started. Or, if you are raring to go, just follow the simple steps for your build tool:

Maven sbt

Using Lagom with Java and Maven

Prerequisites for using Lagom with Maven

  • JDK 8
  • Maven 3.3 or higher

See detailed instructions for help setting up the prerequisites and getting started. Or, if you are raring to go, just follow these simple steps:

  1. Create a new project from the command line (all on one line): 

    mvn archetype:generate -DarchetypeGroupId=com.lightbend.lagom -DarchetypeArtifactId=maven-archetype-lagom-java -DarchetypeVersion=1.3.0
  2. Maven will prompt you for groupId (which becomes the package name), artifactId (which becomes the name of the project’s folder), and version.

  3. Change into the project folder and run it: 

    mvn lagom:runAll

See the documentation to test and understand your new project.

Using Lagom with Java and sbt

Prerequisites for using Lagom with sbt

  • JDK 8
  • sbt 0.13.13

See detailed instructions for help setting up the prerequisites and getting started. Or, if you are raring to go, just follow these simple steps:

Create and run a new project from the command line: 

sbt new lagom/lagom-java.g8 --name=hello
cd hello
sbt runAll

See the documentation to test and understand your new project.
