Using Lagom with Scala

Prerequisites for using Lagom with Scala:

  • JDK 8
  • sbt 0.13.13

If you have the prerequisites installed and are familiar with sbt, follow the steps below. Otherwise, see detailed instructions for help setting up the prerequisites and getting started.

  1. From a command prompt, enter the following to create a new Lagom project: 
    2. sbt new lagom/lagom-java.g8
  2. The sbt Lagom template prompts for the following parameters. Press Enter to accept the defaults or specify your own values:

    • name - Becomes the name of the top-level directory.
    • organization - Used as a package name.
    • version - A version number for your system.
    • lagom-version - The version number of Lagom. Be sure to use the current stable release

    When you finish, sbt downloads all of the project dependencies.

  3. Change into the top-level directory of the project you just created. For example, if you used the default name, hello: cd hello
  4. Start sbt and the Lagom development environment: sbt runAll
  5. Verify that the services are indeed up and running by invoking the hello service endpoint from any HTTP client, such as a browser:

    http://localhost:9000/api/hello/World

    The request returns the message Hello, World!.

Congratulations! You've created and run your first Lagom system. See the documentation to understand your new project.
