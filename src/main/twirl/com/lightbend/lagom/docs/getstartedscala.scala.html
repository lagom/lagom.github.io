Using Lagom with Scala

The Lagom Scala API supports sbt. If you are using Scala, you probably already have sbt installed, but make sure it is the right version.

Prerequisites for using Lagom with Scala:

  • JDK 8
  • sbt 0.13.13

See detailed instructions for help setting up the prerequisites and getting started. Or, if you are raring to go, just follow these simple steps.

Create and run a new project from the command line: 

sbt new lagom/lagom-scala.g8 --name=hello
cd hello
sbt runAll

See the documentation to test and understand your new project.
