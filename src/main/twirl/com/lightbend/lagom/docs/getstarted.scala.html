Get started with
Lagom @context.currentLagomVersion today

Change log

Lagom integrates with industry standard build tools which will download and run everything you need to get started with developing your Lagom system.

Select your language:

Java Scala

Lagom provides a Java API and a Scala API. Both APIs support the full range of Lagom features and provide a great microservice development experience, which language you select is up to you.

Select your build tool:

sbt Maven

You can develop Lagom Java services using either sbt or Maven. Both build tools offer the full range of Lagom features and provide a great microservice development experience. If you don't have an opinion on which build tool you want to use, we recommend using sbt.

Lagom Java API with sbt

Download and install JDK 8.

Download and install sbt 0.13.13.

Create a new project from the command line: 

$ sbt new lagom/lagom-java.g8

Run an existing project from its directory: 

$ sbt runAll

Lagom Java API with sbt

Download and install JDK 8.

Download and install Maven.

Create a new project from the command line: 

$ mvn archetype:generate -DarchetypeGroupId=com.lightbend.lagom \
  -DarchetypeArtifactId=maven-archetype-lagom-java -DarchetypeVersion=1.1.0

Run an existing project from its directory: 

$ mvn lagom:runAll
